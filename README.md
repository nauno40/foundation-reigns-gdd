# Foundation × Reigns

Game Design Document pour **Foundation**, un jeu de cartes narratif à swipe dans l'univers d'Isaac Asimov.

## Concept

Incarner une succession de Speakers de la Seconde Fondation sur 1 000 ans d'histoire galactique. Chaque règne est une partie roguelike — les décisions façonnent l'histoire, la mort est inévitable, mais le Plan de Seldon continue.

**Inspiré de :** [Reigns: Three Kingdoms](https://www.devolverdigital.com/)  
**Moteur :** Godot 4.x (mobile + PC)  
**Univers :** Foundation — Isaac Asimov  
**Langue principale :** Français  

## Documents

- [GDD.md](GDD.md) — Game Design Document complet (v1.0)

## Mécaniques clés

- 4 ressources (Militaire / Religion / Commerce / Politique) — game over à 0 ET à 100
- Légitimité : jauge cachée, signaux textuels uniquement
- 6 Crises de Seldon comme jalons narratifs
- 9 factions · 12 planètes · 41 decks · ~1 160 cartes
- Double progression : score de règne + rang méta (15 rangs)
- Mort naturelle probabiliste 75→83 ans
